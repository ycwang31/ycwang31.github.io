# Yongcong Wang's Personal Website

Source for [ycwang31.github.io](https://ycwang31.github.io/), a lightweight academic and professional profile built with semantic HTML and responsive CSS.

## Local preview

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000/`.

## Main files

- `index.html`: profile content, metadata, and JSON-LD structured data
- `stylesheet.css`: responsive layout and visual design
- `Yongcong-Wang-Resume.pdf`: current resume
- `robots.txt` and `sitemap.xml`: crawler discovery files
